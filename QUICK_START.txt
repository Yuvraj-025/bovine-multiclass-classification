COW BREED CLASSIFIER - QUICK START GUIDE
========================================

Welcome to the Cow Breed Classification Application!

QUICK START:
1. Double-click on "start-all.bat" (Windows) or run "./start-all.sh" (Linux/Mac)
2. Wait for both servers to start (this may take 10-30 seconds)
3. Open your browser and go to http://localhost:3000
4. Upload an image of a cow and select a model to get predictions

SERVER ADDRESSES:
- Backend API: http://127.0.0.1:8000
- Frontend UI: http://localhost:3000

AVAILABLE MODELS:
Models are automatically loaded from the "models/" directory:
- cattle_breed_classifier.pth
- cattle_cnn_model.pth
- cattle_cnn_model_lesslayers.pth
- cattle_cnn_model_more.pth
- resnet50_cattle_classifier.pth

If you add new models to the "models/" directory, click "Reload Models" in the UI
or restart the backend server.

Need help? Check the SETUP.md and API_RESPONSES.md files for detailed instructions.
